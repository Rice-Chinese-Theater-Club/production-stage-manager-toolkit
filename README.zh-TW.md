# Production Stage Manager Toolkit

**Rice 中文話劇社製作工具合集**

---

## 簡介

Production Stage Manager Toolkit 是 Rice 中文話劇社製作舞台監督 (Production Stage Manager) 團隊所使用的數位工具的集中目錄。本儲存庫本身不包含程式碼，而是作為一個導航中心，指向各個獨立的工具儲存庫。

---

## 工具列表

| 工具 | 儲存庫 | 說明 | 狀態 |
|---|---|---|---|
| 電子走位本 | [stage-blocking-app](https://github.com/Rice-Chinese-Theater-Club/stage-blocking-app) | 線上走位記錄與台詞管理系統，支援即時同步 | Active |
| 字幕產生器 | [subtitle-maker](https://github.com/Rice-Chinese-Theater-Club/subtitle-maker) | 雙語字幕 PPT 自動產生工具 | Active |
| AI 助理舞監 | [ai-stage-manager](https://github.com/Rice-Chinese-Theater-Club/ai-stage-manager) | AI 助理舞台監督（開發中） | Planning |
